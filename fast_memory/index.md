# 知识图谱导航

> 本文档是 Agent 认知快记忆区的总目录，帮助快速定位信息归属。

---

## 一、记忆源清单

| 类别 | 文件/目录 | 功能 |
|------|----------|------|
| 系统宪法 | schema.md | Agent 行为规范与安全红线 |
| 纠错覆盖 | corrections.md | 管理员人工修正的事实记录 |
| 高频问答 | public_faq.md | 自动沉淀的脱敏问答对 |
| 实体字典 | entities/ | 部门架构、项目编号等对齐词典 |
| 文档骨架 | docs_index/ | 各 PDF 文件的结构化摘要卡片 |
| 业务技能 | skills/ | 各业务流程的 SOP 操作规范 |

---

## 二、典型查询路径

```
用户: "查询2026年人工智能专项的拨款进度"
  ↓
1. 检查 corrections.md → 是否有关于"人工智能专项"的修正记录
  ↓
2. 检查 public_faq.md → 是否有类似的高频问答
  ↓
3. 匹配技能 → 命中 "资金拨付状态查询" 技能
  ↓
4. 实体消歧 → 到 entities/key_projects.md 查标准项目编号
  ↓
5. 检索 docs_index/ → 定位 2026_AI_Project.pdf 的章节结构
  ↓
6. 向量检索 → 到 zvec 中取相关切片
  ↓
7. 组装回答 → 按技能要求的格式输出
```

---

## 三、知识更新流程

- **发现错误** → 管理员执行 `correct` 命令，追加到 corrections.md
- **高频问题** → 后台审查模块自动脱敏后写入 public_faq.md
- **新文件入库** → 放入 source/ 目录，执行 `index` 命令重建索引
- **业务流程变更** → 修改 skills/ 下对应的 MD 文件，执行 `reload-skills`
